# SUNU DIGITAL - Landing Page

Une landing page moderne et persuasive pour présenter une formation gratuite sur la création de vidéos avec l'IA.

## Fonctionnalités
- Design moderne avec Next.js 14
- Identité visuelle africaine (Vert, Jaune, Rouge)
- Psychologie de conversion intégrée
- Responsive design
- Animations avec Framer Motion
- Icônes avec Lucide-React

## Installation
1. `npm install`
2. `npm run dev`

## Déploiement sur Vercel
1. Pousser le code sur GitHub.
2. Connecter le repo à Vercel.
3. Les variables d'environnement ne sont pas nécessaires pour le moment.

## Structure
- `src/app/page.tsx` : Contenu principal de la landing page.
- `src/app/globals.css` : Design system et tokens.
- `public/` : Logo et images hero.
